Documentation

Source Code

You can find the source code for this project on Github. We built the project using open source materials from Bootstrap, JQuery, Leaflet, OpenStreetsMap, PostgreSQL, and PostGIS.

Data

Below is information about the data contained in the database behind the Plan Integration Tool. IDs and geometry fields are excluded.

Land Use Plans

Attribute Meaning
pln_title The title of the plan.
pln_typ The category of the plan.
auth The authority, such as an agency or organization, who created the plan.
cnty The county location of the plan's boundary.
pln_ar The jurisdiction name which the plan administers.
pln_yr The year the plan was created.
url The URL location of a PDF file of the plan.

Montgomery County Bus Stops

Attribute Meaning
stop_name The agency name of the stop, often the closest intersection cross-streets.
town The jurisdiction in which the stop is located.
stop_code The agency code for the stop.

Prince George's County Bus Stops

Attribute Meaning
variance
shelter Denotes the existence of a bus shelter at the stop location.
receptacle Denotes the existence of a waste receptacle at the stop location.
tmpt_suit
unsf_loc Denotes that the bus stop is in an unsafe location. Criteria unknown.
shrd_stp Provides identifiers for other transit lines with which the stop is shared.
comments
route The agency route designation.
wmata
thebus
notes
name
route_numb The agency route designation.
stop_numbe The agency stop number designation within the route.
street_nam The name of the street on which the stop is located.
street_int The closest intersection to the stop.
bus_direct The cardinal direction of the bus when it visits the stop.
location

WMATA Metro Lines

Attribute Meaning
line The name of the metro line.
year The year the metro line opened.
from_ The name of the stop from which trains on the segment are leaving.
to_ The name of the stop to which trains on the segment are moving towards.
display The display name of the metro line.
anode
bnode
