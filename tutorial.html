Tutorial

Follow the directions below to navigate through the Plan Integration Tool under a specific scenario. Walking through the scenario should help you understand how the tool works, and allow you to customize it to your own needs.

  1. Navigate to the tab labeled Map.
  2. Once the Map page loads, navigate to the panel on the left side of the page. At the top of the panel, there are three buttons: Intro, Query, and Analyze. Select Query.
  3. Once you select Query, you are presented with a form, with three dropdown lists. In the first dropdown list, at the top of the panel, you are prompted to Select a table. You can learn more about the tables in the Documentation page of this website, but for now, choose Land Use Plans from the dropdown list.
  4. Once you select Land Use Plans, a list of selection options will appear below the dropdown list. These selection options are the attributes of Land Use Plans, and you will be using these attributes to narrow down the information that gets displayed on the map. Select the option labeled pln_typ.
  5. Below the list of selection options, there is another dropdown list: Select an operator. Select equal to from the dropdown list.
  6. Below the Select an operator dropdown list is the final dropdown list: Select a value after picking a column. Select Master Plan from the dropdown list.
  7. Finally, select the Submit button at the bottom of the panel.

This completes your search for data: the data that will display on the map are Land Use Plans where the pln_typ (or Plan Type) is equal to Master Plans. You can remix this format using different tables, different attributes, different operators, and different values to find the information that is relevant to you and your plans.
